Let's replace all dogs with cats

:%s/dog/ cat/g

What just happened?
  :%s specifies search and replace
  then we specify old word
using same syntax we learned for search.
  specify new word
and we end with /g for global
to replace all instances of dog with cat.

Tip: if you want to confirm before replacing add 'c' at the end.

You can also search only a specific region of text by visually selecting first, then typing :s/oldword/ newword/g

Let's greet Frank, Buddy, and the rest of the gang using Vim's automation powers. Macros allow you to record and replay any actions:

1. Go to line 23 ("Frank")
2. Start recording by typing qa
  (this tells vim to record in register 'a')
3. Type I to add a "hey" at the begining
4. Exit Insert mode then type A to add an '!' at the end
  (you love Frank)
5. Finally type q to stop recording

You just recorded your first macro. Go down a line to "Buddy" and replay those steps by typing @a

Isn't that amazing? Imagine all the ways you can use macros.

Let's greet our gang of dogs even more efficiently.
