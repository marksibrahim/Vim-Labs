Visual mode highlights text before you operate on it. It's a great way to make sure you're deleting or copying the right thing.

v starts Visual Mode. Now, apply any motion to select text.

Try v + } to grab a paragraph.

w is useful for grabbing a word as is selecting up to a character f + character

For more complex selections, activate Visual Block Mode:
V + any motion. You can use j, k, l to highlight blocks of text.
So what? You can, for example, insert text at the begining of many lines all at once!

try selecting the first character on lines 6-9:
V + j then I and ')'. Once you escape to Normal mode, text will appear on all lines!

Visual selection is useful for copying.

Try visually selecting a paragraph: v + } then copy (yank) using: y.

p pastes the selection. ( P pastes above)
You can also quickly copy lines using yy
You can apply a similar trick to delete a line: dd
Once you delete a piece of text, it's automatically added to your clipboard. So after deleting a word or paragraph you can paste it elsewhere by simply typing p!

Try copying then pasting a word: yaw then p.

With actions operating on objects, comes the power to repeat. Type a number before an action to repeat it.

Try copying a sentence. Remember how? (yy)
Instead of pasting once, type 3p.

Try applying the same idea to a motion. Try going down 10 lines? You could press j ten times or you can type 10j! Finally, if you want to undo not only the last edit, but the last 2. What do you do? 2u!

How do you search for a word in Vim? It's easy! Just type \a word To navigate to the next instance, type n (N to go backwards). Vim shines because by default, you can search not just for words verbatum, but also patterns. Let's search for all words with v_m where _ can be any character: \v.m Find this or that: \this\|that For more regex see here... One other really neat trick is *, searches for the word under teh cursor, so you dont' have to type it!
