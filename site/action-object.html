Vim's syntax is intuitive: Actions operating on text Objects

try typing ' d + a + w ' to delete the word under your cursor

d is the delete Action
a is around
w w is word.

This syntax generalizes to sentences, paragraphs, quotes, and other objects. To delete a paragraph: ' d + a + p '

As you gain Vim superpowers, you'll learn to rely on the keyboard (not the mouse).
  j to move down
  k to move up

your index finger naturally rests on 'j'


You can also navigate paragraphs:
  ] down a paragaph
  [ up a paragraph

down a paragraph = next blank line

Let's kick it up a notch
  G jump to the bottom
  gg jump to the top

Vim understands words, paragraphs, and other text objects

You can navigate words
  w next word (first letter)
  e next word (last letter)

  b previous word (first letter)

You can jump directly to any character on a line
  f + character

from the start of line 7, type 'f' followed by 'V'

You can go backwards using
  F + character

Once finished :w saves your work.
